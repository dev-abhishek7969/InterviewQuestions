HTML stands for HyperText Markup Language. It is used to design the structure of web pages using markup.

A semantic element clearly describes its meaning to both the browser and the developer, such as <article>, <section>, and <footer>.

JavaScript is a programming language used to make web pages interactive and dynamic.

A closure is a function that has access to its outer function's scope even after the outer function has returned.
